# BWM427_Analyzer

Программный модуль на Python для автоматизированной обработки массивов данных, получаемых при проведении опыта кренования.

Главная задача скрипта — исключить рутину и человеческий фактор при математических расчетах посадки и остойчивости, обеспечив быструю и точную выдачу результатов на основе сырых логов.

## Стек технологий
* **Язык:** Python
* **Анализ данных:** Pandas

## Что реализовано в программе
* **Парсинг логов:** Автоматическое чтение и структурирование неразмеченных массивов данных, поступающих с инклинометров.
* **Фильтрация шумов:** Очистка выборки от вибраций, помех и аномальных выбросов (сглаживание значений).
* **Вычислительное ядро:** Точный расчет фактических углов крена на основе отфильтрованных массивов.
* **Экспорт результатов:** Агрегация данных и автоматическое формирование сводных таблиц/отчетов для прикрепления к протоколу испытаний.
